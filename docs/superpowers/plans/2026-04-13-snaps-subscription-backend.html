구독 서비스 에러 코드 (RtCode)

SubscriptionException 발생 시 SubscriptionControllerAdvice가 ApiResponseMessage 형식으로 응답합니다.

에러 응답 포맷

{
  "errorCode": "A500S01",
  "errorMessage": "Active subscription not found.",
  "message": "활성 구독이 없습니다."
}
CodeHTTP설명
A500S00400이미 활성 구독이 존재
A500S01404활성 구독을 찾을 수 없음
A500S02400구독이 일시정지 상태가 아님 (resume 시)
A500S03404결제수단이 등록되지 않음
A500S04500PG 결제 실패
A500S05404포토북을 찾을 수 없음
A500S06400발행된 포토북은 수정 불가
A500S07400커버 텍스트 20자 초과
A500S08404공유 대상자를 찾을 수 없음
A500S09404배송 주문을 찾을 수 없음
A500S10404공유 이력을 찾을 수 없음
A500S11404요금제를 찾을 수 없음

제외된 배치 (별도 프로젝트 권장)

7 Job
플랜이 Spring Batch (Spring Boot 3.x 환경) 기반이므로, 현재 Spring MVC/Java 8 프로젝트에는 포함하지 않았습니다. 기존 snaps-batch 프로젝트 또는 신규 배치 전용 프로젝트에 구현 권장.
